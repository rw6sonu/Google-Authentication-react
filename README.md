 The Firebase Authentication System is a secure, user-friendly web application developed using React.js for the frontend and Firebase Authentication for backend user management. This system enables users to register, log in, and manage their profiles with secure authentication, supporting both email/password and social login methods. Real-time authentication state monitoring ensures a smooth user experience across all devices.

✅ Core Features
User Registration: Sign up with email/password or social providers like Google, Facebook, or GitHub, with optional email verification.
User Login: Secure login with email/password or social login, keeping users authenticated across sessions.
Password Management: Reset forgotten passwords via email and change password while logged in.
Protected Routes / Dashboard: Restrict access to certain pages for authenticated users only, with automatic redirection for unauthenticated users.
User Profile Management: Update display name, profile picture, and other optional user details.
Real-Time Auth State: UI dynamically updates based on whether the user is logged in or logged out.
